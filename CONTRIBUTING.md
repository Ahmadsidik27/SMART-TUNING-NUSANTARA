### Reporting Bugs

1. Search for existing issues to avoid duplicates
2. Create a new issue with:
   - Clear title
   - Detailed description
   - Steps to reproduce
   - Screenshots if applicable
   - Environment information

### Suggesting Features

1. Search for existing feature requests
2. Create a new issue with:
   - Clear title
   - Detailed description
   - Use cases
   - Any relevant mockups
